# index/

This directory stores the vector index used by the retrieval system.

Typical files include:
```bash
- `faiss.index` — FAISS vector index containing embeddings of rule chunks
- `meta.json`   — metadata mapping index entries to original chunks
```
These files are generated by the indexing pipeline:
```bash
rag/build_index.py
```
The index is built from processed chunks located in:
```bash
knowledge/processed/
```
