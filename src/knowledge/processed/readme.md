# processed/

This directory stores processed knowledge artifacts used by the retrieval system.

Typical files include:

- `chunks.jsonl` — chunked representations of rules/examples used for embedding
- `last_retrieval.json` — cached retrieval results produced during pipeline runs

The chunk file is generated by:
```bash
rag/make_chunks.py
```
Chunks are later embedded and indexed by:
```bash
rag/build_index.py
```
